# Transformation and Optimization in Deep Learning for Trajectory Analysis

This repository contains my BSc project on applying transformation and optimization techniques in deep learning for trajectory analysis. The work focuses on analyzing motion trajectories using neural network–based approaches, similarity measures, and feature transformations.

## Overview

Trajectory data arises in many domains such as transportation systems, human activity recognition, surveillance, and sports analytics. This project investigates how deep learning methods can be used to extract meaningful patterns from trajectory data through:

- Feature transformation techniques for trajectory representation  
- Neural network models for trajectory pattern learning  
- Similarity measures (including cosine similarity) for trajectory comparison  
- Performance evaluation using standard classification metrics  

The implementation emphasizes clarity and experimentation with different modeling approaches.
